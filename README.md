# 🎵 Music Player App

## 📌 Project Overview
The **Music Player App** is a simple application that allows users to play, pause, and manage music easily. This project demonstrates basic programming concepts such as user interface design, audio handling, and event-driven programming.

The main goal of this project is to create an interactive and user-friendly music player while practicing programming and application development skills.

---

## 🚀 Features
- ▶️ Play music
- ⏸ Pause music
- ⏭ Next song
- ⏮ Previous song
- 🎚 Volume control
- 📂 Display list of songs
- 🎧 Simple and clean user interface

---

## 🛠 Technologies Used
- Python
- Streamlit / Tkinter
- Pygame (for audio playback)
- Git & GitHub

## ⚙️ Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/music-player-app.git
```

### 2️⃣ Navigate to the project folder

```bash
cd music-player-app
```

### 3️⃣ Install required libraries

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the application

```bash
python app.py
```
